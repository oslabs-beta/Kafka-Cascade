# static service(kafka, topic, groupId, serviceCB, successCB, dlqCB) → {CascadeService}

Main entry point to the module. Creates a new service that listens to and produces kafka messages
Parameters:
Name Type Description
kafka KakfaJS Kafka Object
topic string Topic that the service listens for and runs the service
groupId string Group Id for the service consumer
serviceCB Callback that is run whenever 'topic' is received or retry. It accepts a kafka message, resolve callback and reject callback
successCB Callback that is run when the serviceCB resolves a message, accepts the kafka message
dlqCB Callback that is run when the serviceCB rejects a message and cannot be retried anymore
